# E-Commerce and Inventory Management System

This project is a simple *E-Commerce and Inventory Management System* developed using *Java*.

## Features
- Manage customers
- Manage products
- Create and track orders
- Handle transactions
- Basic data management structure

## Technologies Used
- Java
- HTML 
- CSS3 and javaScript

## Project Structure
- Customer.java – Represents customer information
- Product.java – Represents product details
- Order.java – Manages customer orders
- Transaction.java – Handles transactions
- DataManager.java – Manages data operations
- index.html – Interface for the system

## How to Run
1. Clone or download the repository
2. Open the project in a Java IDE (IntelliJ, Eclipse, NetBeans)
3. Compile and run the Java files
